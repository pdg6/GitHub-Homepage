Beginner Level Examples

1. Text Styling with HTML & CSS

Concept: HTML provides the text content, while CSS adds visual styling.

Bold and Colored Text Italic Text Underlined Text Uppercase with Spacing
<!-- HTML --> <p style="color: blue; font-size: 20px;">Styled Text</p> /* CSS */ p { color: blue; font-size: 20px; font-weight: bold; }

How it works: The <p> tag creates a paragraph. CSS properties like color, font-size, and font-weight control appearance.

2. Interactive Button with JavaScript

Concept: JavaScript makes elements respond to user clicks.

<!-- HTML --> <button onclick="showMessage()">Click Me!</button> <script> function showMessage() { alert('Hello! You clicked the button!'); } </script>

How it works: The onclick attribute calls a JavaScript function when clicked. The alert() function displays a popup message.

3. Simple Counter

Concept: JavaScript can store values in variables and update the page dynamically.

0
<div id="counter">0</div> <button onclick="incrementCounter()">Add 1</button> <script> let count = 0; function incrementCounter() { count++; // Add 1 to count document.getElementById('counter').textContent = count; } </script>

How it works: A variable count stores the number. When clicked, count++ increases it by 1. getElementById finds the element, and textContent updates what's displayed.

Novice Level Examples

4. Form with Validation

Concept: JavaScript can check user input before submission.

<form onsubmit="return validateForm(event)"> <input type="text" id="username" placeholder="Name" required> <input type="email" id="email" placeholder="Email" required> <button type="submit">Submit</button> </form> <script> function validateForm(event) { event.preventDefault(); // Prevent page reload const username = document.getElementById('username').value; const email = document.getElementById('email').value; if (username && email) { alert('Form submitted successfully!'); return true; } return false; } </script>

How it works: event.preventDefault() stops the form from reloading the page. JavaScript checks if fields have values using .value, then provides feedback.

5. Animated Progress Bar

Concept: Combine CSS transitions with JavaScript to create smooth animations.

0%
<div class="progress-bar"> <div class="progress-fill" id="progress">0%</div> </div> /* CSS */ .progress-fill { width: 0%; transition: width 0.5s ease; background: linear-gradient(90deg, #667eea, #764ba2); } <script> function fillProgress() { let progress = 0; const interval = setInterval(() => { progress += 10; document.getElementById('progress').style.width = progress + '%'; document.getElementById('progress').textContent = progress + '%'; if (progress >= 100) clearInterval(interval); }, 200); } </script>

How it works: setInterval() runs code repeatedly every 200ms. Each time, it increases progress by 10% and updates the bar's width. clearInterval() stops at 100%.

Intermediate Level Examples

7. Tab Navigation System

Concept: Switch between different content panels without page reload.

HTML - The Structure

HTML provides semantic structure using tags like <header>, <nav>, <section>, and <footer>. These elements organize content logically.

CSS - The Design

CSS uses selectors to target elements and apply styles. Modern CSS includes Flexbox and Grid for powerful layouts, plus animations and transitions for smooth effects.

JavaScript - The Behavior

JavaScript uses variables, functions, conditionals (if/else), loops, and DOM manipulation to create dynamic experiences. Modern JS includes async/await for handling delays and API calls.

<div class="tabs"> <button class="tab" onclick="switchTab(event, 'tab1')">Tab 1</button> <button class="tab" onclick="switchTab(event, 'tab2')">Tab 2</button> </div> <div class="tab-content active" id="tab1">Content 1</div> <div class="tab-content" id="tab2">Content 2</div> <script> function switchTab(event, tabId) { // Hide all content document.querySelectorAll('.tab-content').forEach(content => { content.classList.remove('active'); }); // Remove active from all tabs document.querySelectorAll('.tab').forEach(tab => { tab.classList.remove('active'); }); // Show selected content and activate tab document.getElementById(tabId).classList.add('active'); event.currentTarget.classList.add('active'); } </script>

How it works: querySelectorAll() finds all matching elements. forEach() loops through them. We remove 'active' from everything, then add it to the clicked tab and its content.

Modal Dialog Example

This is a modal dialog! It overlays the page and can contain any content - forms, images, videos, or information.

Modals are useful for:

  • Image galleries
  • Important notifications
<button onclick="openModal()">Open Modal</button> <div id="modal" class="modal"> <div class="modal-content"> <span class="close-modal" onclick="closeModal()">×</span> <h2>Modal Title</h2> <p>Modal content here</p> </div> </div> /* CSS */ .modal { display: none; position: fixed; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0,0,0,0.7); } .modal.active { display: flex; } <script> function openModal() { document.getElementById('modal').classList.add('active'); } function closeModal() { document.getElementById('modal').classList.remove('active'); } </script>

How it works: The modal is hidden by default (display: none). position: fixed makes it overlay the page. Adding the 'active' class changes display to 'flex', centering the content.

9. Responsive Card Grid

Concept: CSS Grid creates flexible layouts that adapt to screen size, with hover effects.

üöÄ Feature 1

Responsive design adapts to any screen size automatically.

‚ö° Feature 2

Smooth animations and transitions enhance user experience.

üéØ Feature 3

Interactive elements respond to user actions instantly.

üîí Feature 4

/* CSS Grid Layout */ .card-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 2rem; } .card { padding: 2rem; border-radius: 10px; box-shadow: 0 4px 6px rgba(0,0,0,0.1); transition: all 0.3s ease; } .card:hover { transform: translateY(-10px); box-shadow: 0 10px 20px rgba(0,0,0,0.2); }

How it works: display: grid creates a grid layout. repeat(auto-fit, minmax(250px, 1fr)) automatically fits cards, making them at least 250px but expanding to fill space. :hover applies styles on mouse over.

10. Dynamic To-Do List

Concept: Create, display, and remove items dynamically with JavaScript DOM manipulation.

    <input type="text" id="todo-input" placeholder="Enter a task"> <button onclick="addTodo()">Add Task</button> <ul id="todo-list"></ul> <script> function addTodo() { const input = document.getElementById('todo-input'); const todoText = input.value.trim(); if (todoText === '') { alert('Please enter a task!'); return; } // Create new list item const li = document.createElement('li'); li.textContent = todoText; li.style.cssText = 'padding: 0.8rem; margin: 0.5rem 0; background: #f0f0f0; border-radius: 5px; cursor: pointer;'; // Remove on click li.onclick = function() { this.remove(); }; // Add to list document.getElementById('todo-list').appendChild(li); input.value = ''; // Clear input } </script>

    How it works: document.createElement('li') creates a new element. .textContent sets its text. .style.cssText applies multiple styles. appendChild() adds it to the page. .remove() deletes it.
